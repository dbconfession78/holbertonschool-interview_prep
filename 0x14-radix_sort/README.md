# 0x14. Radix Sort

## Description
This application will sort an integer array using the LSD version of Radix Sort

## Compilation
```
gcc -Wall -Wextra -Werror -pedantic 0-main.c 0-radix_sort.c print_array.c -o radix
```

## Usage
`./radix

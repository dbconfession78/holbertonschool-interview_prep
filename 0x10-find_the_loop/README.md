# 0x10. Find the Loop

## Description
This application finds which node a linked list begins a loop

## Compilation
```
gcc -Wall -pedantic -Werror -Wextra 0-find_loop.c -L. -lloop -o find_loop
```

## Usage
`./find_loop`
